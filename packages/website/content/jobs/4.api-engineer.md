##### What is Otutu?

Otutu is an accounting platform featuring full double-entry accrual accounting, delivering modern automation for professional financial management. We're building the next generation of financial technology solutions that help businesses and individuals manage their finances with precision and automation.

---

##### How your work with us will look like?

You will be responsible for designing, building, and maintaining robust API integrations with various financial institutions, payment processors, and third-party services using Laravel. You'll work closely with our engineering team to ensure scalable, secure, and reliable API infrastructure that supports our accounting automation features.

This role involves deep technical work with financial APIs, implementing Laravel Sanctum/Passport for authentication, building comprehensive API documentation, and ensuring data integrity across multiple integrations.

---

##### Technical Requirements:

- **Laravel Expertise:**
  - Advanced Laravel framework experience (8+ years)
  - RESTful API development with Laravel
  - Laravel Sanctum/Passport for API authentication
  - API resource classes and API versioning
  - Laravel API rate limiting and throttling

- **Database & Performance:**
  - MySQL/PostgreSQL optimization for API queries
  - Database indexing strategies for API performance
  - Laravel Eloquent API resource optimization
  - Caching strategies for API responses

- **Security & Compliance:**
  - OAuth 2.0 implementation and management
  - JWT token handling and validation
  - API security best practices (CORS, CSRF protection)
  - Financial data encryption and secure transmission
  - PCI DSS compliance for payment integrations

- **Financial API Integration:**
  - Payment processor APIs (Stripe, PayPal, Flutterwave)
  - Banking API integrations
  - Financial data standardization
  - Error handling and retry mechanisms
  - Webhook implementation and management
