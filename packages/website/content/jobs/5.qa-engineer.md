##### What is Otutu?

Otutu is an accounting platform featuring full double-entry accrual accounting, delivering modern automation for professional financial management. We're building the next generation of financial technology solutions that help businesses and individuals manage their finances with precision and automation.

---

##### How your work with us will look like?

You will be responsible for ensuring the quality and reliability of our Laravel/Filament accounting platform through comprehensive testing strategies. You'll work closely with our engineering and product teams to develop test plans, execute test cases, and identify potential issues before they reach our customers.

This critical role involves testing financial calculations, Laravel API endpoints, Filament admin interfaces, database integrity, and ensuring compliance with accounting standards. You'll be safeguarding financial data accuracy that businesses depend on.

---

##### Technical Requirements:

- **Testing Frameworks & Tools:**
  - PHPUnit for backend testing (Laravel applications)
  - Pest PHP testing framework experience
  - Laravel Dusk for browser testing
  - Experience with API testing tools (Postman, Insomnia)
  - Database testing and migration testing

- **Financial Software Testing:**
  - Double-entry bookkeeping validation testing
  - Financial calculation accuracy testing
  - Multi-currency transaction testing
  - Tax calculation and compliance testing
  - Accounting report generation testing

- **Quality Assurance Skills:**
  - Test case design and execution
  - Regression testing and automated test suites
  - Bug tracking and reporting (Jira, GitHub Issues)
  - Performance testing for financial operations
  - Security testing for financial data protection

- **Laravel/Filament Specific:**
  - Filament admin panel testing
  - Livewire component testing
  - Eloquent model relationship testing
  - Laravel Sanctum/Passport authentication testing
  - Queue and job testing
