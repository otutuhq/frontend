##### What is Otutu?

Otutu is an accounting platform featuring full double-entry accrual accounting, delivering modern automation for professional financial management. We're building the next generation of financial technology solutions that help businesses and individuals manage their finances with precision and automation.

---

##### How your work with us will look like?

You will be the frontline technical support for Otutu's Laravel/Filament accounting platform, helping users resolve technical issues, providing guidance on platform features, and ensuring smooth operation. You'll work closely with our engineering team to understand system updates and communicate effectively with accountants, bookkeepers, and business users.

This role involves troubleshooting Laravel/Filament applications, guiding users through complex workflows, and maintaining detailed documentation for common issues and solutions.

---

##### Technical Requirements:

- **Laravel/Filament Knowledge:**
  - Solid Laravel framework understanding (routing, middleware, controllers)
  - Filament admin panel navigation and functionality
  - Basic Eloquent ORM operations and relationships
  - Laravel authentication and user management
  - Understanding of Laravel application structure

- **Troubleshooting Skills:**
  - Browser developer tools and frontend debugging
  - Network request analysis and API troubleshooting
  - Database query understanding and basic SQL knowledge
  - Log file analysis and error message interpretation
  - Environment configuration and deployment issues

- **Financial Software Support:**
  - Accounting workflow guidance and best practices
  - Financial report generation troubleshooting
  - Data import/export issue resolution
  - Multi-user access and permission management
  - Backup and data recovery assistance

- **Support Tools & Documentation:**
  - Helpdesk ticketing system management (Zendesk, Intercom)
  - Knowledge base creation and maintenance
  - User guide development and updates
  - Video tutorial creation and hosting
  - Remote support tools (screen sharing, session recordings)
