##### What is Otutu?

Otutu is an accounting platform featuring full double-entry accrual accounting, delivering modern automation for professional financial management. We're building the next generation of financial technology solutions that help businesses and individuals manage their finances with precision and automation.

---

##### How your work with us will look like?

You will be developing new features across our full stack, integrating with various financial APIs, building robust backend systems with Laravel, and creating intuitive admin interfaces with Filament. There's also always a lot of troubleshooting user reported issues, bugs, and feature requests. You can improve our current Laravel/Filament implementation and help us get Otutu to the next level!

As a growing platform, we work collaboratively on our codebase and take pride in building professional, scalable solutions. This is an exciting opportunity to shape the future of financial automation using modern PHP frameworks.

---

##### Technical Requirements:

- **Backend Excellence:**
  - Advanced Laravel framework experience (8+ years preferred)
  - Eloquent ORM mastery and complex query optimization
  - Laravel Nova or similar admin panel experience (Filament preferred)
  - RESTful API development with Laravel Sanctum/Passport
  - Queue systems (Redis, database queues) and job processing
  - Laravel caching strategies and performance optimization

- **Database & Architecture:**
  - MySQL/PostgreSQL database design and optimization
  - Database migration management and schema versioning
  - Understanding of database indexing and query performance
  - Experience with database backup and recovery strategies

- **Frontend & Admin Panels:**
  - **Filament PHP** - Advanced experience required
  - Livewire component development
  - Alpine.js integration with Filament
  - Tailwind CSS for custom styling
  - Responsive design principles

- **DevOps & Deployment:**
  - Docker containerization
  - CI/CD pipeline experience
  - Server deployment and management
  - SSL certificate management
  - Performance monitoring and optimization
