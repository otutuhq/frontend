##### What is Otutu?

Otutu is an accounting platform featuring full double-entry accrual accounting, delivering modern automation for professional financial management. We're building the next generation of financial technology solutions that help businesses and individuals manage their finances with precision and automation.

---

##### How your work with us will look like?

You will be responsible for the deployment, monitoring, and maintenance of our Laravel/Filament accounting platform infrastructure. You'll ensure high availability, performance, and security of our production environment while working closely with our development team to streamline the deployment process.

This role involves managing PHP applications, database administration, server optimization, and implementing best practices for scalable Laravel deployments in a financial technology environment.

---

##### Technical Requirements:

- **Laravel Deployment & Infrastructure:**
  - Laravel application deployment and configuration
  - PHP-FPM optimization and server tuning
  - Laravel queue management and worker processes
  - Laravel Horizon monitoring and job management
  - Environment-specific configuration management

- **Server & Database Administration:**
  - Linux server administration (Ubuntu/Debian)
  - MySQL/PostgreSQL database administration and optimization
  - Nginx/Apache web server configuration for PHP
  - SSL certificate management and HTTPS enforcement
  - Server security hardening and firewall management

- **DevOps & Automation:**
  - Docker containerization for Laravel applications
  - CI/CD pipeline management (GitHub Actions, GitLab CI)
  - Infrastructure as Code (Terraform, Ansible)
  - Automated deployment scripts and rollback procedures
  - Monitoring and alerting system setup

- **Cloud & Performance:**
  - AWS/DigitalOcean cloud infrastructure management
  - Load balancing and auto-scaling configuration
  - Database backup and disaster recovery
  - Performance monitoring and optimization
  - CDN configuration and static asset management
