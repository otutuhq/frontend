##### What is Otutu?

Otutu is an accounting platform featuring full double-entry accrual accounting, delivering modern automation for professional financial management. We're building the next generation of financial technology solutions that help businesses and individuals manage their finances with precision and automation.

---

##### How your work with us will look like?

You will be responsible for implementing and maintaining comprehensive security measures across our Laravel/Filament accounting platform. You'll focus on protecting sensitive financial data, ensuring compliance with financial regulations, and implementing security best practices throughout the development lifecycle.

This critical role involves Laravel application security, financial data protection, API security, and ensuring our platform meets the highest security standards required for financial technology.

---

##### Technical Requirements:

- **Laravel Security Expertise:**
  - Advanced Laravel security implementation (Sanctum, Passport, custom guards)
  - Laravel encryption and data protection strategies
  - Secure authentication and authorization systems
  - Laravel middleware for security controls
  - Secure API design and implementation

- **Application Security:**
  - OWASP Top 10 mitigation strategies for Laravel applications
  - SQL injection prevention and database security
  - Cross-site scripting (XSS) and CSRF protection
  - Secure file upload and storage practices
  - Input validation and sanitization

- **Financial Data Security:**
  - PCI DSS compliance for payment processing
  - Financial data encryption at rest and in transit
  - Secure key management and cryptography
  - Audit trail implementation for financial transactions
  - SOX compliance and financial reporting security

- **Infrastructure Security:**
  - Server hardening and secure deployment practices
  - SSL/TLS certificate management
  - Secure Docker container configurations
  - Web application firewall (WAF) implementation
  - Security monitoring and logging
