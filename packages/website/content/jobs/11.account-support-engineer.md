##### What is Otutu?

Otutu is an accounting platform featuring full double-entry accrual accounting, delivering modern automation for professional financial management. We're building the next generation of financial technology solutions that help businesses and individuals manage their finances with precision and automation.

---

##### How your work with us will look like?

You will be responsible for providing technical support to our customers using Otutu's Laravel/Filament accounting platform. You'll help users troubleshoot issues, resolve complex technical problems, and ensure they get the most value from our platform. You'll work closely with our engineering team to understand system behavior and communicate effectively with accountants and bookkeepers.

This role involves deep Laravel/Filament knowledge, database troubleshooting, API issue resolution, and providing clear solutions to users with varying technical backgrounds.

---

##### Technical Requirements:

- **Laravel/Filament Expertise:**
  - Strong Laravel framework knowledge (8+ years)
  - Filament admin panel troubleshooting and customization
  - Eloquent ORM query optimization and debugging
  - Laravel Sanctum/Passport authentication issues
  - Queue system monitoring and job troubleshooting

- **Database & Performance:**
  - MySQL/PostgreSQL query optimization and debugging
  - Database migration and seeding issue resolution
  - Performance bottleneck identification and resolution
  - Data integrity and consistency troubleshooting
  - Backup and recovery support

- **Financial Application Support:**
  - Double-entry bookkeeping workflow troubleshooting
  - Financial calculation accuracy verification
  - Multi-currency and tax calculation support
  - Accounting report generation issue resolution
  - Financial data import/export troubleshooting

- **Technical Support Skills:**
  - API integration troubleshooting and configuration
  - Third-party service integration support
  - Browser compatibility and frontend debugging
  - Server log analysis and error investigation
  - Environment-specific issue diagnosis
